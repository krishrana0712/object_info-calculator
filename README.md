# 📐 Object Info Calculator (Geometric Engine)

A modular C-based computational tool designed to calculate the geometric properties (Area, Perimeter, Volume) of 2D and 3D objects. This project focuses on translating complex mathematical formulas into structured, algorithmic logic.

## 🚀 Key Features
* **Modular Architecture:** Functions are separated by shape (Sphere, Cone, Quadrilaterals, etc.) for high maintainability.
* **Coordinate Geometry:** Implementation of advanced formulas for complex polygons.
* **Persistent Interface:** Built with a nested `switch-case` menu and `do-while` loops for a seamless user experience.
* **Precision Control:** Uses `double` precision and formatted output (`%.2f`) for professional-grade accuracy.

## 🛠️ Technical Stack
* **Language:** C (Standard C11)
* **Environment:** VS Code / GCC
* **Core Libraries:** `stdio.h`, `math.h`

## 📂 Repository Structure
* `object_info_calculator.c`: Main source code containing the logic engine.
* `.gitignore`: Configured to exclude compiled binaries (.exe) for a clean repository.
